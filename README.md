# Actividad 04 - Modulo 04

## Descripción

Este es un proyecto de JavaScript donde se manipulan elementos del DOM (Document Object Model) usando 4 tareas diferentes.

## Las 4 Tareas

### Tarea 1: Repetir Palabra
- Ingresa una palabra y cuántas veces quieres repetirla
- El programa repite la palabra en pantalla

### Tarea 2: Aplicar Color
- Elige un color con el selector
- El texto cambia al color que elegiste

### Tarea 3: Calcular
- Ingresa dos números
- El programa calcula: suma, resta, multiplicación y división

### Tarea 4: Invertir Texto
- Escribe un texto cualquiera
- El programa lo invierte al revés

## Archivos

- `index.html` - Estructura HTML
- `style.css` - Estilos CSS
- `script.js` - Funciones JavaScript
